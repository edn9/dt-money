# DT-Money
Estudo de App do 3º Projeto do módulo [React Native](https://app.rocketseat.com.br/classroom/projeto-aplicacao-financeira-2), Aplicação financeira com sistema de login, cadastro de usuario, criação de transações monetarias com filtragem e acesso a API backend local.

# Como Rodar Aplicação
Projeto feito visando android. Use SDK 17 LTS, recomendo [SDKMAN](https://sdkman.io/) para versionamento java. Se o script não funcionar, baixe zip e unzip no chocolatey (terminal admin):

```bash
sdk install java 17.0.14.fx-zulu 
```

```bash
sdk default java 17.0.14.fx-zulu
```

Instalar as libs:
```bash
npm install
```   
2. Rodar via comando android:
   
```bash
npm run android
```

Rodando backend:
```bash
npm run dev
```
